# Aplicación de Recuperación Aumentada por Generación (RAG)

Este repositorio contiene un notebook de Google Colab que guía al usuario en la construcción de una aplicación de Recuperación Aumentada por Generación (RAG). La aplicación combina modelos de lenguaje avanzados con sistemas de recuperación de información para responder preguntas basadas en datos específicos.

Esta aplicación permite preguntar a un modelo del lenguaje basado en la tecnología de openai preguntas específicas sobre contenido de un post publicado en [datos.gob.es](https://datos.gob.es).

## Contenido del Repositorio

- **Notebook de Google Colab**: Proporciona una implementación detallada de una aplicación RAG, incluyendo:
  - Carga y procesamiento de datos.
  - Integración con modelos de lenguaje como GPT-4.
  - Configuración de sistemas de recuperación de información.
  - Generación de respuestas basadas en datos recuperados.

## Requisitos

Antes de utilizar el notebook, asegúrate de contar con:

- Una cuenta activa en [OpenAI](https://openai.com) con una clave de API válida.
- Una cuenta activa en [LangChain](https://smith.langchain.com/) con una clave de API válida.
- Acceso a Google Colab para ejecutar el notebook.
- Conocimientos básicos de Python y procesamiento de lenguaje natural.

## Cómo utilizar el Notebook

1. **Acceso**: Abre el notebook en Google Colab utilizando el enlace proporcionado en este repositorio.
2. **Configuración**: Sigue las instrucciones para instalar las dependencias necesarias y configurar las claves de API requeridas.
3. **Ejecución**: Ejecuta las celdas del notebook secuencialmente para construir y probar la aplicación RAG.
4. **Personalización**: Modifica el código según tus necesidades para adaptarlo a diferentes fuentes de datos o modelos de lenguaje.
